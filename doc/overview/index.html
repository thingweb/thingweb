Thingweb

Exploratory implementation of the WoT Model

Created by Johannes Hund / @h0ru5 and the thingweb community

Some Facts

  • MIT licensed
  • to be public hosted (* currently bitbucket, to be moved to github)
  • Language: Java 8
  • Gradle build, CI server (drone.io), downloadable artifacts

Components

  • Thing description parser
  • Shared base classes / toolkit
  • Server-side framework
  • Client-side framework
  • Example GUI client
  • example servient for the LED pilot

What is the plugfest demo showing?

  • a servient that reads in a TD and attaches application callbacks
    (onInvoke/onUpdate)
  • generic client GUI based that reads in a TD file and interacts with the corresponding thing
  • Interaction with microcontroller-based implementations (ESP8266 + LUA)
  • Protocols: CoAP & HTTP, JSON payloads
